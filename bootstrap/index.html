Jumbotron

Bootstrap allows you to add a variation of a header called "Jumbotron" As you can see above the big blue rectangle with text inside is exactly that.

Colors

Bootstrap has pre made color fields you can use to easier style your page making it look very clean and appealing. Some examples of color fields are : Primary, Success, Danger, Warning, Light, Dark, etc. Feel free to google the others!

CSS Grids

Bootstrap using CSS Grids makes it super easy to layout your site and align all the elements. The recent update makes it easier than ever before.
